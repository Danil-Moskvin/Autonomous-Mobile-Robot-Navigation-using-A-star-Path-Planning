# Autonomous Mobile Robot Navigation using A* Path Planning

## Описание проекта

Данный проект реализует систему автономной навигации всенаправленного мобильного робота с использованием алгоритма поиска пути **A*** и системы компьютерного зрения на основе ArUco-маркеров.

Робот перемещается по заранее подготовленной карте, содержащей свободную область, препятствия и конечную точку. Текущее положение робота определяется камерой, расположенной сверху, после чего алгоритм A* строит безопасную траекторию движения. Управление осуществляется посредством пропорционального регулятора (P-регулятора), а вычисленные команды преобразуются в скорости отдельных колес с помощью обратной кинематики всенаправленной платформы.

# Возможности

* Построение кратчайшего маршрута с использованием алгоритма A*.
* Локализация робота по ArUco-маркеру.
* Управление всенаправленной платформой.
* Объезд статических препятствий.
* Автоматическое движение к заданной цели.
* Визуализация карты и построенного маршрута.
* Логирование экспериментальных данных.
* Построение графиков ошибок, скорости и траектории движения.

# Принцип работы

1. Загружается карта рабочей области.
2. Определяются свободные области, препятствия и конечная цель.
3. Камера определяет текущее положение робота по ArUco-маркеру.
4. Алгоритм A* строит оптимальный маршрут до цели.
5. P-регулятор вычисляет управляющее воздействие относительно следующей точки маршрута.
6. Обратная кинематика преобразует команды движения в скорости четырех колес.
7. ESP32 получает команды и управляет двигателями.
8. Во время движения выполняется постоянное обновление координат робота и корректировка управления.
9. После достижения цели сохраняются результаты эксперимента и строятся графики.

# Обозначения карты

* **Черный цвет** — свободная зона движения.
* **Красный цвет** — препятствия.
* **Зеленая точка** — конечная цель.

# Запуск проекта

1. Подключить ESP32 и камеру.
2. Поместить ArUco-маркер на робота.
3. Подготовить карту и сохранить её в папку `experiment/`.
4. Указать IP-адрес ESP32 в файле `main.py`.
5. Запустить проект:

```bash
python main.py
```
